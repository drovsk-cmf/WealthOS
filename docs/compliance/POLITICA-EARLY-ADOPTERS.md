# Oniefy — Política de Early Adopters

**Versão:** 1.0
**Data:** 25 de março de 2026
**Válida para:** usuários cadastrados antes do lançamento público

---

## 1. Compromisso

O Oniefy reconhece que os primeiros usuários assumem riscos ao confiar seus dados financeiros a um produto novo. Em troca, oferecemos condições permanentes e irreversíveis.

## 2. Quem é considerado Early Adopter

Qualquer pessoa com conta criada antes da data oficial de lançamento público (a ser definida e comunicada com 30 dias de antecedência).

## 3. Benefícios garantidos

| Benefício | Detalhe |
|-----------|---------|
| **Acesso vitalício ao plano vigente** | O plano e os limites ativos na data de cadastro são mantidos para sempre, mesmo que novos planos sejam criados com preços diferentes. |
| **Preço congelado** | O valor da assinatura (quando implementada) nunca aumenta para early adopters, mesmo com reajustes futuros. |
| **Acesso antecipado a funcionalidades** | Novas features em beta são liberadas primeiro para early adopters. |
| **Canal direto de feedback** | Acesso ao canal de comunicação direta com o fundador para sugestões e bugs. |
| **Crédito de fundador** | Nome (opcional) listado na página "Fundadores" do produto, se autorizado. |

## 4. O que NÃO fazemos

- **Não removemos features já entregues.** Se você usava uma funcionalidade antes de uma mudança de plano, ela permanece acessível.
- **Não forçamos migração de plano.** Se surgir um plano "Premium" com features novas, early adopters não perdem o que já tinham.
- **Não usamos dados para marketing.** Dados financeiros são seus. Não vendemos, não compartilhamos, não usamos para anúncios.

## 5. Transparência sobre limitações

- O produto está em desenvolvimento ativo. Bugs existem e serão corrigidos.
- Funcionalidades podem mudar de nome ou interface, mas a capacidade será preservada.
- Caso uma feature seja descontinuada (raro), early adopters serão notificados com 90 dias de antecedência e terão opção de exportar dados relacionados.

## 6. Validade

Esta política é permanente e irrevogável para quem se qualifica como early adopter. Alterações futuras se aplicam apenas a novos usuários.

---

*Inspirado nos erros do Organizze (mudança retroativa de plano) e nos acertos do YNAB (grandfathering consistente). A confiança dos primeiros usuários é o ativo mais valioso do Oniefy.*
